# Oracle Database Backup and Recovery Management System

This project demonstrates practical Oracle Database Administration tasks including RMAN backup, recovery management, Flashback Query, and database monitoring using Oracle Database XE and SQL Developer.

## Technologies Used
- Oracle Database XE
- RMAN
- Oracle SQL Developer
- SQL / PL SQL

## Features
- Full database backup using RMAN
- Database recovery operations
- Flashback Query implementation
- Data loss simulation and recovery
- Backup verification and monitoring
